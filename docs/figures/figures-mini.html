From-scratch figures

Two publication figures for the pure-NumPy companion to CheXVision — a neural network built by hand: forward pass, backprop, optimizers, all without a framework or autograd. Same visual system as the parent figure family — restrained slate-blue feature pathway, coral reserved for the backward pass and the correctness proof. Inter + JetBrains Mono throughout.

Figure 1End-to-end pipeline. Top lane: each X-ray is streamed from Hugging Face, downsampled to 64×64 grayscale, flattened to 4096 numbers, and standardized. Bottom lane: the per-mini-batch loop — MLP forward, BCE-with-logits, hand-derived backprop, an Adam + cosine step — repeated, with the best epoch kept after a validation pass. Pure NumPy, CPU only.
Figure 2Architecture & hand-derived backprop. The MLP compresses 4096 inputs through three Linear · ReLU · Dropout blocks to a single logit, then a sigmoid + BCE-with-logits loss. The coral pass returns the gradient by the chain rule — entry gradient (σ(z) − y)/N, and per-layer dW = xᵀ·dout, db = Σ dout, dx = dout·Wᵀ — every one derived by hand and confirmed against a finite-difference check to < 1e-6.
