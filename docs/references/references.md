# References

## ref1 — Neural Networks from Scratch in Python

- **Authors:** Harrison Kinsley (Sentdex) & Daniel Kukieła
- **Publisher / source:** self-published, https://nnfs.io
- **ISBN:** 978-1-7363-5750-9
- **Used for:** the from-scratch derivations in this repo — dense layers,
  activation functions, binary cross-entropy, backpropagation via the chain
  rule, and the optimizer update rules (SGD, momentum, Adam).

> The book (`ref1_NNSP.pdf`) is a copyrighted commercial work and is read
> locally during development. Obtain your own copy from https://nnfs.io.
